## Overview

**Page Purpose:** Immediate feedback confirming the log entry, with discreet optional actions (add note, add weight). Auto-saves after 3 seconds if user doesn't interact — the "Option C" pattern.

**User Situation:** Camille just selected an action + target. She sees the confirmation and either ignores it (auto-save) or enriches the entry with a note or weight. She's already mentally moving on to her next activity.

**Success Criteria:**
- Confirmation visible within 200ms of selection
- Auto-save in 3 seconds if no interaction
- Optional enrichment (note/weight) accessible but never required
- Total flow time: < 3 seconds for frequent actions

**Entry Points:**
- 02.2 Quick Log — Sélection (animal/resource selected)
- Auto-skip from 02.1 (single animal, no resources)

**Exit Points:**
- Auto-dismiss after 3 seconds → back to Dashboard (updated)
- Tap note action → expand note input, then save → Dashboard
- Tap weight action → expand weight input, then save → Dashboard
- Tap undo → cancel entry, back to Dashboard

---

## Design Decision: Option C — Auto-Enregistrement

The confirmation toast appears with a 3-second countdown. Optional actions (note, weight) are visible as ghost buttons. If ignored, the entry auto-saves and the toast slides away. If the user taps an option, the countdown stops, the toast expands, and the user enriches the entry before manual save.

**Why:** Camille's primary mode is "log and forget." Requiring explicit confirmation adds friction to the most frequent action. The options are there for the 20% of cases where she wants to add detail.

## Technical Notes

- **Auto-save timing:** 3 seconds is calibrated — long enough to see options, short enough that Camille doesn't wait. If user starts interacting, countdown stops entirely.
- **Offline support:** Entry is saved to local storage immediately. Sync happens in background on reconnect.
- **Haptic feedback:** Light haptic on auto-save completion.
- **Keyboard handling:** When note/weight input opens, ensure toast doesn't get pushed behind keyboard. Toast should reposition above keyboard.
- **Weight input:** Decimal separator adapts to locale (. for EN, , for FR).
- **Undo window:** Even after auto-save, keep undo available for 5 more seconds in the "Saved ✓" mini-toast.
