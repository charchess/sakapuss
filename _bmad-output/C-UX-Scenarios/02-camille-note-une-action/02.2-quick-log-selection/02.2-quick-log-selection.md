## Overview

**Page Purpose:** Camille selects the target (animal or household resource) for her action. For frequent actions, this is the ONLY intermediate step — friction proportional to frequency.

**User Situation:** Camille just tapped an action tile on the Dashboard. She knows WHAT she did, now she picks WHO/WHAT it applies to. If she has only one animal and no shared resources for this action, this step auto-skips.

**Success Criteria:**
- Selection completed in < 1 tap (auto-skip) or 1 tap (pick animal/resource)
- No unnecessary choices presented
- Smart defaults reduce cognitive load

**Entry Points:**
- Dashboard quick action tile tap (with `action` pre-selected)
- Bottom nav "+" center button (no action pre-selected — show category picker first)

**Exit Points:**
- Select target → 02.3 Quick Log — Confirmation
- Auto-skip (single animal, no shared resources) → 02.3 directly
- Swipe down / tap backdrop → dismiss, back to Dashboard

---

## Design Decision: Friction Proportionnelle

| Action Frequency | Steps | Example |
|------------------|-------|---------|
| **Daily** (litter, food) | 2 taps total | Tile → animal → auto-confirm |
| **Weekly** (weight) | 3 taps + value | Tile → animal → enter value → confirm |
| **Occasional** (free event) | Short form | Tile → animal → type text → confirm |

**Auto-skip rule:** If user has 1 animal AND the action doesn't involve shared resources (caisses, gamelles) → skip selection, go directly to confirmation.

## Technical Notes

- **Auto-skip logic:** Check `user.animals.length === 1` AND `action.requires_resource === false` → skip to 02.3.
- **Resource model:** Resources are household-level entities named by admin. API returns `household.resources` filtered by action type.
- **Bottom sheet:** Height adapts to content. Max 60% viewport. Snap points: collapsed (dismiss) ↔ expanded.
- **Haptic:** Light haptic on animal/resource selection for tactile confirmation.
- **Performance:** Pre-load animals and resources when Dashboard mounts — selection sheet must appear instantly.
