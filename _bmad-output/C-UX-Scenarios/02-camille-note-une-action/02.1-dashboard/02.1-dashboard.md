## Overview

**Page Purpose:** Entry point for the log-first experience — Camille sees her quick action grid immediately on app open and can log an action in one tap.

**User Situation:** Camille just did something for her cats (cleaned litter, filled bowl, noticed something unusual). She grabs her phone to log it before she forgets. Standing in the kitchen or living room, one hand free.

**Success Criteria:**
- Time to first tap on an action: < 1 second after app opens
- Most frequent actions visible without scrolling
- Active reminders visible but non-blocking

**Entry Points:**
- PWA shortcut on home screen (primary — 90% of entries)
- Bottom navigation "Home" tab (from other app sections)
- Push notification tap → redirects here if no specific target

**Exit Points:**
- Tap quick action tile → 02.2 Quick Log — Sélection
- Tap center "+" button → 02.2 Quick Log — Sélection (expanded)
- Tap reminder banner → 03.1 Liste des rappels
- Tap animal avatar → 04.3 Profil animal
- Bottom nav → Timeline, Reminders, Profile

## Page States

| State | When | Appearance | Actions |
|-------|------|------------|---------|
| Default | Has animals + some activity | Full grid + recent entries | All actions available |
| Empty — First Visit | Account created, no logs yet | Grid visible + empty recent with prompt "Log your first action!" | Grid active, reminder if configured |
| Empty — No Animals | Arrived without completing onboarding | Prompt to add first animal | "Add your first animal" CTA → 01.2 Ajout animal |
| Loading | App opening / data sync | Skeleton grid (6 placeholder tiles) + skeleton recent | Wait |
| Offline | No network connection | Full UI from cached data, subtle offline indicator | All local actions work, sync on reconnect |

## Technical Notes

- **Performance:** Dashboard must render in < 300ms. Pre-cache grid tiles and last 3 entries.
- **PWA:** Service worker caches grid config. Quick actions work offline, sync on reconnect.
- **Animal tabs:** If only 1 animal, hide tabs — grid shows all actions directly.
- **Grid order:** Most frequent actions first (tracked per user). Default order: Caisse, Gamelle, Pesée, Médicament, Observation, Libre.
- **Reminder banner:** Max 1 visible at a time, swipe to see next if multiple. Priority: overdue > today > upcoming.
