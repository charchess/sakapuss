## Overview

**Page Purpose:** Optional configuration wizards organized by category: Health reminders (Sante), Initial weight (Poids), Food resources (Alimentation). Each category is entirely skippable. Camille configures health reminders for her cats, skips the rest, and lands on the Dashboard feeling that Sakapuss is already working for her.

**User Situation:** Camille just added Pixel and Luna. She is invested — her animals are in the system. But fatigue risk is real: this is the third step of onboarding. Every screen must justify its existence. If it feels like a chore, she will skip everything.

**Success Criteria:**
- At least 1 reminder configured (primary success metric for BG0)
- Each category completable in < 20 seconds
- Skip affordance always visible — never trapped
- User can return to skipped categories later from Settings

**Entry Points:**
- After adding animals (01.2) — automatic redirect
- From Settings → "Complete setup" (if skipped earlier)

**Exit Points:**
- "Finish" / "Go to Dashboard" → 02.1 Dashboard
- Each category skip → next category or finish

## Technical Notes

- **Species-driven suggestions:** Reminder suggestions are filtered by species. Cats get: deworming, flea treatment, annual vaccine. Dogs get the same plus: annual vet checkup, heartworm. "Other" species get a blank list with "Add custom reminder" prominent.
- **Pre-checked strategy:** Deworming and flea treatment are pre-checked by default (highest value reminders). This nudges Camille toward configuring at least 1 reminder — the BG0 activation metric.
- **All categories skippable.** The user can skip every single category and reach the Dashboard. Skipped categories are accessible later from Settings (01.5).
- **Reminder creation:** Toggling a reminder ON creates it with the default frequency. The user can adjust frequency later from the reminder detail view.
- **Start date:** Reminders start from today. If user knows when the last treatment was, they can edit the reminder later to set a custom start date.
- **Category transition animation:** Horizontal slide (current slides left, next slides in from right). Progress bar animates segment fill.
- **Completion celebration:** Confetti-style micro-animation with checkmark. "You're all set!" message. Auto-redirect to Dashboard after 2 seconds (tappable to go immediately).
- **Persistence:** Each category saves independently. If user quits mid-wizard, completed categories are saved. Uncompleted categories remain available from Settings.
- **Offline:** Category data can be configured offline. Saved on reconnect.
