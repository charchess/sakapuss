## Overview

**Page Purpose:** Account settings hub — profile, language, notification preferences, household management, and access to skipped onboarding categories. Lighter spec: settings page with grouped sections.

**User Situation:** Camille accesses settings from the Profile tab or after onboarding to fine-tune notifications or revisit skipped setup steps.

**Success Criteria:**
- All settings reachable within 2 taps from Dashboard
- Skipped onboarding categories clearly surfaced as "Complete setup"
- Language switch is instant (no page reload)

**Entry Points:**
- Bottom navigation → Profile tab → Settings icon/link
- Onboarding completion → optional "Customize settings" link

**Exit Points:**
- Back arrow → previous page (Profile or Dashboard)
- Tap any setting group → detail view

## Technical Notes

- **Language switch:** Updates i18n locale in memory. All visible text re-renders immediately. Persisted to user profile on server.
- **Notification preferences detail view:** Push on/off, reminder lead time (1 day, 3 days, 1 week), daily digest toggle, quiet hours.
- **Setup banner persistence:** Disappears once all onboarding categories are completed. Categories track completion status individually.
- **Log out:** Clears JWT, local cache, and service worker data. Redirects to Login.
